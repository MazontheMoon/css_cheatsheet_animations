CSS Animations

  • Animations
  • Keyframes

CSS animations have two parts, the first using the animation property to specify how the animation sequence progresses and then using @keyframes rule for the visual appearance of the animation.

Animations

  • animation-name
  • animation-duration
  • animation-timing-function
  • animation-delay
  • animation-iteration-count
  • animation-direction
  • animation-fill-mode
  • animation-play-state
  • animation

animation-name

...name of the @keyframes rule.

.box:hover {
animation-name: rotate;
} /*multiples are allowed*/
.box:hover {
animation-name: rotate, slide;
} .box:hover {
animation-name: none;
/*none is a special keyword denoting no keyframes and can deactivate an animation*/
}

animation-duration

...length of time for 1 cycle of the animation sequence.

.box:hover {
animation-name: rotate;
animation-duration: 2s;
/*set in s or ms (or -s -ms)*/
}

animation-delay

...delay from time it is loaded to the start of animation.

.box:hover {
animation-name: rotate;
animation-duration: 2s;
animation-timing-function: ease;
animation-delay: 1s; /*can be set as s, ms, -s , or -ms (negative -1s will start 1 sec into animation) and default is 0*/
}

animation-iteration-count

...number of times the animation repeats.

.box:hover {
animation-name: rotate;
animation-duration: 2s;
animation-timing-function: ease;
animation-delay: 1s; animation-iteration-count: 3; /*values are numbers (1.5 will run 1 and a half times) or infinite*/
}

animation-play-state

...Set to pause and resume the animation sequence.

.box:hover {
animation-name: rotate;
animation-duration: 2s;
animation-timing-function: ease;
animation-direction: alternate-reverse;
animation-fill-mode: both;
animation-play-state: paused;
OR animation-play-state: running;
}

animation

...shorthand for all above.

.box:hover {
animation-name: rotate;
animation-duration: 2s;
animation-timing-function: ease;
animation-delay: 1s; animation-iteration-count: 3; animation-direction: alternate-reverse;
animation-fill-mode: both;
animation-play-state: paused;
}

or...

.box:hover {
animation: rotate 2s ease 1s 3 alternate-reverse both paused;
}

Keyframes

Keyframes control the steps in animation sequence by defining styles for keyframes (or waypoints) along the sequence.

Keyframes use % to indicate the time (0% is start and 100% is end). Can also use keyword from for 0% and to for 100%

@keyframes slidein {
from {
transform: translateX(0%);
}

to {
transform: translateX(100%);
}

}

or....
@keyframes slidein {
0%{
transform: translateX(0%);
}

50%{
transform: translateX(50%);
}

100% {
transform: translateX(100%);
}

}
